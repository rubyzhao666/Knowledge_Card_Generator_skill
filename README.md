# 知识卡片快速生成器

将知识内容自动拆解为适合小红书/朋友圈刷屏传播的知识卡片。

## 🎯 功能特点

- **知识拆解**：自动从长文本中提取核心知识点、关键概念
- **模块化输出**：6大核心模块（主题引入、核心概念、要点拆解、实践方法、误区提醒、金句总结）
- **多风格适配**：极简清新、科技商务、温暖治愈、知识趣味
- **平台优化**：针对小红书/朋友圈传播优化

## 📥 输入

- 知识内容文本（可直接粘贴）
- 目标受众（职场新人/学生/从业者/小白用户）
- 卡片风格偏好
- 输出形式（单张/系列）
- 传播平台

## 📤 输出

- 竖版知识卡片（3:4比例）
- 6个核心信息模块
- 可视化图解
- 金句总结

## 🚀 使用示例

**输入：**

> 把"番茄工作法"做成一张知识卡片，适合职场新人看，发小红书

**输出：**

- 主标题：番茄工作法 一张卡讲清
- 模块1：主题引入 - 为什么高效工作如此重要
- 模块2：核心概念 - 25分钟专注+5分钟休息
- 模块3：要点拆解 - 4个番茄钟后长休息
- 模块4：实践方法 - 3步开始番茄工作法
- 模块5：误区提醒 - 别在番茄钟内看手机
- 模块6：金句总结 - 专注是高效的第一生产力
